# Bryson's Utah Road Ready

A mobile-first, single-page study app for the Utah written driver knowledge test.

## Features
- Adaptive question selection weighted toward missed questions and weak categories
- 50-question simulated exam mode
- Topic practice and missed-question review
- Local browser storage for progress, XP, streaks, scores, and readiness
- Personalized encouragement for Bryson
- Readiness estimate based on accuracy, topic coverage, and recent sessions
- No server, login, database, or external API required

## Source basis
Official Utah Driver Handbook 2025–2026, revision 3.2026, plus the Utah DLD Written Knowledge Test page.

This is an unofficial study aid and not affiliated with the Utah Driver License Division.
